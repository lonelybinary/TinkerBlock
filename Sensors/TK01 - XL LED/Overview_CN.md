# 功能

本模块是一个蓝色 LED 模块，可以通过程序控制 LED 的亮灭。LED 是一种发光二极管，通电后会发出蓝色光，可以用来做指示灯、装饰灯等。

模块上有一个蓝色 LED 灯珠和一个 4 针排针接口，可以通过丝印（引脚旁边印刷的文字）识别每个引脚。

# 引脚

- **GND**（负极）：就像电池的负极（-），连接到控制板的 GND
- **NC**（空引脚）：无实际电路连接，为统一接口故作此引脚，可悬空不接
- **NC**（空引脚）：无实际电路连接，为统一接口故作此引脚，可悬空不接
- **SIGNAL**（信号输入）：控制 LED 亮灭的引脚，连接到控制板的数字引脚（如 Arduino 的 D13 或 Pico 的 GPIO 0）
  - 高电平（HIGH/1）时 LED 点亮
  - 低电平（LOW/0）时 LED 熄灭

**注意：** 本模块有两个 NC 引脚，都是空引脚，可以悬空不接。

# 特点

- 蓝色 LED，亮度适中
- 工作电压：3.3V 或 5V
- 低功耗：电流很小

# 快速接线

1. GND → 控制板 GND
2. SIGNAL → 控制板数字引脚（程序中用哪个引脚就接哪个）
3. NC 引脚可悬空不接
